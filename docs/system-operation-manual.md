# 內部稽核管考追蹤系統操作手冊

只保留常用操作。

## 角色

- `最高管理員`：帳號管理、檢核表管理、教育訓練名單管理、稽核軌跡、版本治理
- `單位管理員`：矯正單、檢核表、教育訓練、資安窗口

## 常用功能

- 登入與密碼重設
- 矯正單：開單、回填、審核、追蹤、結案
- 內稽檢核表：填報、草稿、送出
- 教育訓練：名單、暫存、送出、退回更正、CSV
- 帳號管理：新增、編輯、刪除
- 登入紀錄：查閱

## 使用規則

- 草稿可續填
- 正式送出後不可直接修改
- 角色與單位會限制可見範圍
- 換電腦不會自動同步 localStorage 資料

## 常見問題

- 畫面不正常：先重新整理，再看 hash 是否正確
- 看不到資料：先檢查角色與單位
- 不能修改教育訓練單：看是否已正式送出
- 換電腦看不到資料：目前前端資料存於瀏覽器 localStorage
