# Daily UAT Checklist

## Unit Admin

- Log in with a unit-admin account and confirm the dashboard, checklist, and training entries all load.
- Open the corrective-action list and verify same-unit cases are visible while cross-unit cases are not.
- Start a checklist draft, save it, and confirm the draft can be reopened from the list.
- Open a same-unit training draft created by a reporter and confirm the manual row is visible but not deletable by non-owner.
- Complete training flow step 1, print the signoff sheet, upload the signed file, and confirm the dashboard moves the unit into the completed section.

## Reporter

- Log in with a reporter account and confirm admin-only pages are blocked.
- Open a same-unit checklist draft and submit it successfully.
- Start a training draft, add one manual row with unit, identity, and job title, and save the draft.
- Reopen the same training draft and verify the manual row can still be edited by the creator.
- Confirm duplicate yearly training forms are blocked and redirect to the existing form.

## Upload Safety

- Try uploading an invalid extension file and confirm the system rejects it.
- Try uploading a zero-byte file and confirm the system rejects it.
- Try uploading an oversized file and confirm the system rejects it.
- Upload a valid file, then upload the same file again and confirm duplicate upload is blocked.
