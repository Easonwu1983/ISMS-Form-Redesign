內部稽核管考追蹤系統
小白使用教學 SOP

適用對象：第一次接觸系統的行政同仁、單位填報窗口、主管。

目標：照著流程操作即可完成填報、送出、審核與追蹤，不需要技術背景。

版本：v1.0｜產出日期：2026-03-05｜系統名稱：內部稽核管考追蹤系統

1. 先看這裡（3 分鐘上手）

  1. 打開系統首頁並登入。
  2. 從左側主選單選你要的功能：
    • 矯正單列表 / 開立矯正單
    • 內稽檢核表
    • 教育訓練時數統計
  3. 填報過程可暫存（教育訓練、檢核表草稿）。
  4. 正式送出後通常會鎖定，若要修改需管理者退回。
提醒 本系統不同角色可見功能不同。若你看不到某頁，多半是「角色權限」問題，不是系統壞掉。

2. 登入與密碼重設

2.1 登入

  1. 輸入帳號與密碼。
  2. 按「登入系統」。
  3. 成功後進入儀表板。

2.2 忘記密碼

  1. 按「忘記密碼？」。
  2. 輸入帳號綁定的電子郵件。
  3. 系統會產生新密碼（畫面顯示）。
  4. 回登入頁使用新密碼。

3. 角色與權限對照

角色可做事項
最高管理員 帳號管理、登入紀錄、檢核表管理、教育訓練名單管理、退回更正、全校統計檢視
單位管理員 開立矯正單、填報檢核表、填報教育訓練時數、查看單位資料
填報人 教育訓練時數填報、被指派矯正單之回填作業

4. SOP-A：開立矯正單（單位管理員以上）

  1. 進入「開立矯正單」。
  2. 確認「提出單位」：系統會預設你的單位。
  3. 選擇「處理單位」後，再選「處理人員」。
    處理人員會自動只顯示該處理單位的人，避免選錯。
  4. 填寫缺失分類、來源、問題描述與預定完成日期。
  5. 按「送出矯正單」。

常見錯誤

  • 沒選缺失種類或來源
  • 分類一項都沒勾
  • 處理人員未選，導致無法送出

5. SOP-B：回填矯正措施（處理人員）

  1. 進入矯正單詳情，按「回填矯正措施」。
  2. 填寫：矯正措施、根因分析、根因消除措施。
  3. 必要時上傳佐證（圖片/PDF）。
  4. 送出後狀態會改為「已提案」。

6. SOP-C：審核與追蹤（管理者）

  1. 在「已提案」案件按「進入審核」。
  2. 審核中可選：通過結案 / 轉追蹤 / 退回重填。
  3. 若轉追蹤，可持續記錄追蹤結果（系統最多 3 次追蹤）。
若案件狀態已改變，系統會阻止舊畫面重複送出，請回到詳情頁重新操作。

7. SOP-D：內稽檢核表填報

  1. 進入「填報檢核表」。
  2. 逐題選符合狀態，填寫執行情形與佐證說明。
  3. 可按「儲存草稿」先離開；完成後再「送出檢核表」。

重要規則

  • 自評日期變更時，稽核年度會自動換算成民國年。
  • 草稿可續填；正式送出後不開放直接改內容。

8. SOP-E：教育訓練時數統計

8.1 填報人流程

  1. 進入「教育訓練時數統計」→「新增填報」。
  2. 依名單填每位人員時數與備註。
  3. 若名單缺人，可新增名單外人員。
  4. 可先「儲存暫存」。
  5. 正式送出前，必須上傳簽核掃描檔（圖片/PDF）。

8.2 管理者流程

  1. 在「名單管理」依單位匯入人員名單。
  2. 可刪除名單人員（填報人不可刪）。
  3. 可查看全校填報進度與達標率圖表。
  4. 若誤送，可把「正式送出」資料退回更正。

8.3 匯出

  • 支援全體統計匯出 CSV
  • 支援單一填報單明細匯出 CSV

9. SOP-F：帳號與登入紀錄（管理者）

  • 帳號管理：新增、編輯、刪除使用者
  • 登入紀錄：查看登入成功/失敗（保存最近 500 筆）
  • 可手動清除登入紀錄

10. 常見問題排除

Q1：點某頁會卡住或空白

  • 先重新整理頁面再操作。
  • 確認網址是否正確（例如 `#training`、`#checklist`）。

Q2：我看不到某些資料

  • 多數是角色/單位權限限制。
  • 請請管理者確認你的角色設定。

Q3：為什麼不能改資料

  • 正式送出後資料通常會鎖定。
  • 需要管理者退回，才可再次編輯。

11. 每月建議例行作業（管理者）

  1. 匯出教育訓練 CSV 留存。
  2. 檢查登入紀錄是否有異常失敗嘗試。
  3. 檢查逾期矯正單並追蹤單位改善。
  4. 抽查內稽檢核表填報品質。
