# ISMS 矯正措施追蹤系統

管考追蹤系統 — 基於 ISMS 規範的矯正措施需求單管理系統。

## 功能特色

- 📋 **6 階段工作流程**：開立 → 待矯正 → 已提案 → 審核中 → 追蹤中 → 結案
- 🏷️ **完整 ISMS 分類**：缺失種類、來源、分類（多選）、條文引用
- 🔍 **根因分析 & 風險管理**：Root Cause Analysis + 風險管理（選填）
- 📊 **儀表板**：狀態分布圖、逾期統計、最近矯正單
- 👥 **三種角色**：最高管理員、單位管理員、填報人
- 📧 **通知功能**：開單時可發送信件通知處理人員
- 📎 **佐證上傳**：支援圖片、PDF 佐證文件
- 🔄 **多輪追蹤**：最多 3 輪追蹤監控
